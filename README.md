# ATLAS PSU

Automated Teaching Load Assignment System for Department Chairpersons of the College of Sciences at Palawan State University.

## Features

- Faculty workload balancing
- Teaching load assignment
- Faculty management
- Analytics dashboard
- Load distribution reports

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
```
